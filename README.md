## Google Colab

Run video analysis in the cloud without local setup.

### Quick Start

1. [Open the notebook in Colab](https://colab.research.google.com/github/fuadnafiz98/video-core/blob/master/video_core.ipynb)

2. Run all cells or:
   - Clone repo
   - Install dependencies
   - Upload video or mount Google Drive
   - Run analysis
   - Download results

### Features on Colab

- No local installation required
- Free GPU acceleration (for future features)
- Process large videos in the cloud
- Access videos from Google Drive
